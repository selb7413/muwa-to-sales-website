# MUWA To Sales Website

MUWA sales website extracted into an independent repository.

## Contents

- `index.html` - public sales page
- `styles.css` - page styling
- `script.js` - storefront interactions and product rendering
- `config.js` - public runtime endpoints
- `assets/` - logo, hero images, QR code, and product visuals
- `admin.html`, `admin.js`, `product-admin.html` - admin helpers
- `apps-script-product-admin.gs`, `google-apps-script.js` - Google Apps Script helpers
- `CNAME` - custom GitHub Pages domain

## Deployment

This is a static website and can be deployed with GitHub Pages from the repository root.

Current custom domain:

```text
muwa-to-sales.com
```
